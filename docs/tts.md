# 语音播报

## 简介

语音播报使用系统内置的 TTS（文本转语音）引擎，在点名结束后自动朗读被选中的姓名或小组名。无需联网，完全在本地运行。

## 基本设置

### 开关
勾选「启用语音播报」即可开启。开启后会显示播报参数面板，关闭后所有参数将隐藏但不丢失。

### 语速
控制朗读速度，范围 0.5x（慢速）到 2.0x（快速），默认 1.0x。

### 音调
控制朗读音高，范围 0.5（低沉）到 2.0（尖锐），默认 1.0。

### 音量
控制播报音量，范围 0%（静音）到 100%（最大），默认 100%。

## 自定义播报模板

您可以自定义播报内容，让 TTS 朗读更符合课堂场景。

### 模板变量
使用 `%name%` 作为姓名占位符，播报时会被替换为实际的抽取结果。

**限制**：
- 有效字数上限 **50 字**（`%name%` 算 1 个字）
- `%name%` 最多出现 **3 次**
- 超出限制时输入框会自动截断，并有黄色提示

### 示例
| 模板 | 抽取「张三」时播报 |
|------|-------------------|
| `%name%` | 张三 |
| `请%name%回答问题` | 请张三回答问题 |
| `恭喜%name%被选中` | 恭喜张三被选中 |

### 快速模板
提供 6 个预设模板：默认、课堂提问、恭喜选中、请起立、请上台、组长检查。点击即可应用，当前激活的模板会高亮显示。

### 插入按钮
点击「插入 %name%」按钮可在光标位置插入变量，自动检测字数和变量上限。

## 批量点名播报

批量点名时，多个姓名会用顿号（、）连接后一次性播报。例如抽取了张三、李四、王五，播报内容为模板中 `%name%` 替换为「张三、李四、王五」。

## 测试播报

点击「测试播报」按钮可以用示例姓名「张三」试听当前设置效果（语速、音调、音量、模板均生效）。测试前会先停止正在进行的播报。

## 注意事项

- 语音播报依赖系统 TTS 引擎，不同操作系统的发音效果可能不同
- 每次新播报开始前会自动停止上一次播报，避免重叠
- 如果系统不支持语音合成，播报功能将不可用
- TTS 引擎本身已经通过浏览器的 SpeechSynthesis API 提供，本软件不会替换或修改系统语音引擎
