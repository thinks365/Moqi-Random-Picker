# 帮助：数据管理

## 一、名单管理

### 1.导入名单
- 支持 **Excel**（`.xlsx`、`.xls`）和 **TXT** 文件导入

  #### EXCEL名单的格式：

  | 姓名   | 小组   |
  | ------ | ------ |
  | 李华   | 第一组 |
  | 张三   | 第一组 |
  | 李四   | 第二组 |
  | ...... | ...... |

  #### TXT名单格式

  李华[第一组]

  张三[第一组]

  李四[第二组]

  ......

**（一行一个姓名+组别，组别需用`[]`包裹，TXT文件中不得有空格、空行。）**

### 导出记录（实验性功能）
- 支持导出点名记录为 Excel 文件
- 可选择导出个人点名记录和小组点名记录

## 照片管理

### 添加照片
- 点击「选择照片文件夹」，选中包含照片的文件夹
- 照片文件名需与名单中的姓名匹配
- 支持常见图片格式（JPG、PNG等）

### 注意事项
- 照片仅在通用设置-「点名时显示照片」开关开启时生效
- 仅个人模式支持照片显示

## 音乐设置

- 可以添加点名时播放的背景音乐，提升点名趣味性。
