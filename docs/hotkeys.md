# 快捷键设置

## 总开关

启用或禁用全局快捷键功能。关闭后所有快捷键方案将停止响应，但方案配置会被保留。

## 自定义方案

最多可设置 **5 个** 快捷键方案，每个方案绑定一个组合键。

### 如何设置
1. 点击「添加方案」按钮或点击已有方案的编辑图标
2. 在弹出的对话框中按下你想要使用的组合键
3. 点击「确认」保存

### 支持的按键
- **修饰键**：`Ctrl`、`Shift`、`Alt`、`Win/Cmd`
- **功能键**：`F1`-`F12`、`PageUp`、`PageDown`、方向键等
- **媒体键**：音量控制、媒体播放控制等
- **字母/数字**：必须与修饰键组合使用

### 注意事项
- 快捷键方案与系统其他应用共享，请避免与其他软件的快捷键冲突
- 设置后可在主界面直接按快捷键触发快速点名
