# 帮助：通用设置

## 点名模式

- **个人模式**：随机抽取单个姓名，适用于单人点名场景。
- **小组模式**：随机抽取整个小组，需要先在名单中设置分组信息。

## 显示设置

- **点名时显示照片**：开启后，点名时会同时展示该人员的照片。照片需事先在数据管理中导入。

## 点名设置

- **批量抽取数量**：批量点名时每次抽取的人数（2-50 人/组）。
- **快速点名时间**：快速点名模式下，每个名字显示的时长（0.1-5 秒）。
- **抽选速度**：控制点名动画滚动的速度，1 最慢，5 最快。
- **启用精确点名**：通过加强抽取时间的随机性以提升点名随机性，减少连续抽到相同的名字频率。

## 不重复点名设置

- **启用不重复模式**：开启后，已点过的名字在当前轮次内不会再次被抽到，直到所有名字都点完一轮。

## 硬件加速

- 启用 GPU 硬件加速可提升渲染性能，但可能与部分设备不兼容。更改后需重启应用生效。

## 低功耗模式

- 窗口隐藏时降低资源占用，适合长时间后台运行的场景。
