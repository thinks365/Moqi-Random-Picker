Core Features

软件特性解析

基于全新架构底层打造，各项核心功能全面对齐您的使用场景，让点名更加得心应手。

个人/小组 双模式

支持个人模式与小组模式，且支持主界面一键无缝切换，轻松满足不同形式的课堂互动需求。

悬浮窗便捷点名

还在为频繁切换应用发愁？无需退出PPT放映页面，使用桌面悬浮窗即可一键快捷完成点名。

全局快捷键支持

无需频繁走到电脑前操作。支持设置至多5套快捷键（如 Ctrl+P 等），按一下翻页笔即可触发点名。

便捷的名单管理

支持 TXT / Excel 双模式导入名单。多套名单一并入库管理，走班制频繁切换班级再也无需麻烦。

TTS语音与音效

随机点名也可以妙趣横生！支持调用系统 TTS 引擎自动播报抽取结果，并可配置专属点名背景音乐。

开机自启动设置

告别每次上课前繁杂的手动启动步骤。设置开机自启动程序，或仅静默启动悬浮窗，开机即用。

还有更多功能等你体验：数据防篡改、点名显示照片持续探索中...
