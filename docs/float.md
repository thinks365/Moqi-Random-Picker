# 帮助：悬浮窗设置（实验性功能）<beta-badge></beta-badge>

## 背景设置

- 支持 GIF 和 PNG 格式的背景图片
- 推荐尺寸为 **160×80** 像素的倍数（如 320×160、480×240）
- 点击「清除」可恢复默认透明背景

## 自定义样式

此功能为 <beta-badge></beta-badge> 测试阶段，可能存在显示异常。

- **按钮背景色**：调整悬浮窗按钮的颜色
- **按钮文字色**：调整按钮上文字的颜色
- **名字文字色**：调整点名结果文字的颜色
- **字体大小**：调整悬浮窗中的文字大小

### 注意事项
- 如遇显示问题，请尝试点击「重置样式」
- 若应用样式无效，请先重置后再重新设置
