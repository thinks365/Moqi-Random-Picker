## 二、隐私政策

### 1. 信息收集与存储

​	本软件高度重视用户隐私。我们遵循"数据最小化"原则，**所有数据均存储在您的本地设备上**，不会上传至任何远程服务器。

#### 1.1 本地存储的数据

​	以下数据仅存储在您的设备本地（通过 localStorage 和 electron-store），**不会自动发送给开发者或任何第三方**：

| 数据类别 | 具体内容 | 存储位置 |
|---------|---------|---------|
| 名单数据 | 您导入或创建的姓名列表、分组数据 | localStorage（渲染进程） |
| 点名记录 | 随机点名的历史记录 | localStorage（渲染进程） |
| 应用设置 | 语言偏好、主题选择、快捷键配置、悬浮窗位置、自动启动设置、点名速度、背景图片路径、音量设置等 | electron-store（主进程） + localStorage（渲染进程） |
| 密码哈希 | 您在设置密码保护时生成的 SHA-256 哈希值 | localStorage（渲染进程） |
| 临时状态 | 剩余待点名名单、当前选中名单 ID、模式类型等运行时状态 | localStorage（渲染进程） |

#### 1.2 用户主动导入的数据

- **Excel 文件导入**：通过 SheetJS 库在本地解析，文件内容不会上传至任何服务器。
- **照片文件夹**：通过系统对话框选择的本地文件夹路径，照片通过 `media://` 自定义协议在本地加载显示。
- **背景音乐文件**：用户选择的本地音频文件路径。
- **背景图片**：用户选择的本地图片文件路径。

### 2. 网络通信

#### 2.1 更新检查

​	本软件的唯一网络请求是检查版本更新：

- **请求地址**：`https://rrc.thinks365.com/updates/update.json` 或本软件的github仓库。
- **请求内容**：仅获取版本信息 JSON 文件，**不包含任何用户个人信息、名单数据或设备标识符**。
- **请求方式**：HTTP GET 请求，使用 axios 库。
- **触发时机**：
  - 用户手动点击"检查更新"按钮。
  - 用户在设置中开启了"自动检查更新"功能，软件启动时静默检查。

#### 2.2 更新文件下载

- 如果用户选择自动更新，软件将从更新 JSON 中的下载链接下载安装包。下载过程中 **不会上传任何用户数据**。
- 如果用户选择手动更新，软件将打开系统默认浏览器访问本网站官方网站下的下载页面。此时浏览器行为受浏览器自身的隐私政策约束。

#### 2.3 第三方 CDN 资源加载

​	本软件的主界面（index.html）在启动时会从以下 CDN 加载资源：

- **Font Awesome**（图标库）：从 `cdnjs.cloudflare.com` 加载 CSS 和字体文件。
- **SheetJS (xlsx)**（Excel 处理库）：从 `cdn.sheetjs.com` 加载 JS 文件。

​	这些 CDN 请求是由您的浏览器引擎（Chromium）直接发起的，本软件不会向这些 CDN 发送任何额外数据。CDN 服务商可能收集标准的 HTTP 请求信息（如 IP 地址、User-Agent），这受各 CDN 服务商自身的隐私政策约束。

### 3. 数据安全

#### 3.1 本地安全

- 所有数据存储在您的设备上，使用 Electron 框架提供的标准存储机制（electron-store 使用 JSON 文件存储，localStorage 使用浏览器引擎的本地存储）。
- 密码保护功能使用 SHA-256 算法将密码哈希后存储在 localStorage 中。**请注意**：SHA-256 哈希存储在本地，且哈希值在 localStorage 中可被访问，因此密码保护功能仅作为**温和的访问阻碍**，并非严格的加密安全措施。
- 本软件**不会**对本地存储的数据进行加密。如果您需要在公共或共享设备上使用本软件，请注意数据可能被有权限访问该设备的其他用户查看。

#### 3.2 网络安全

- 更新检查请求使用 HTTPS 加密传输（通过 axios）。
- 更新文件下载可能使用 HTTPS 或 HTTP，取决于更新服务器提供的下载链接。

### 4. 用户权利

​	您对自己的数据拥有完全的控制权：

- **查看权**：所有数据存储在您的设备本地，您可以随时通过软件界面查看。
- **删除权**：您可以在软件设置中执行"重置软件"操作，清除除语言设置外的所有本地数据。您也可以手动删除以下目录中的数据文件：
  - Windows：`%APPDATA%/moqi-random-picker/`
- **选择权**：您可以在设置中选择是否开启自动检查更新功能。

### 5. 儿童隐私

​	本软件本身不收集任何用户的个人信息。如果作为教学工具在课堂中使用，学生的姓名数据完全存储在教师的本地设备上，由教师（数据控制者）负责管理。我们建议教育工作者遵守当地关于学生数据保护的法律法规。

### 6. 隐私政策更新

​	我们可能会不时更新本隐私政策。更新后的隐私政策将通过软件更新同步发布。建议您定期查看本隐私政策以了解最新信息。
